# Localee — бэкенд (API)

Сервер для аккаунтов, друзей и чатов. Стек: **Node.js + Express + SQLite**.

## Запуск локально

```bash
cd server
npm install            # один раз — установить зависимости
cp .env.example .env   # один раз — создать файл настроек
npm run dev            # запустить сервер (перезапускается сам при изменениях)
```

Сервер поднимется на `http://localhost:4000`. Проверка: открой `http://localhost:4000/api/health` — должно вернуть `{"ok":true}`.

База данных — это файл `data/localee.db`, он создаётся автоматически при первом запуске.

## Маршруты (на сейчас)

| Метод | Путь | Что делает |
|-------|------|------------|
| GET  | `/api/health`        | проверка, что сервер жив |
| POST | `/api/auth/register` | регистрация (`name`, `email`, `handle`, `password`) |
| POST | `/api/auth/login`    | вход (`email`, `password`) |
| GET  | `/api/auth/me`       | кто я (нужен токен в заголовке `Authorization: Bearer ...`) |
